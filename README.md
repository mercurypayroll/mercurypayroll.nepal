## ➕ Adding Features Later

All modules are standalone HTML files. To add features:

- **New module** → create new .html file in /modules/
- **New tax slab** → edit slabs[] array in tds-wizard.html
- **New bank** → add bank-item div in payment-gateway.html
- **New employee field** → add form-group in payroll-software.html
- **Change branding** → edit :root CSS variables at top of each file
- **Connect real backend** → replace static JS data with fetch() API calls
