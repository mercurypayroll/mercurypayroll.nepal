Mercury Payroll REST API

Build payroll integrations, automate compliance reporting, sync employees from your HRMS, and trigger payroll runs programmatically. All responses in JSON. Nepal IRD-compliant.

https://api.mercurypayroll.com.np/v2Copy

Format

REST · JSON · HTTPS only

Auth

Bearer token (JWT)

Calendar

BS & AD both supported

Quick start
# Get your API key from Settings → Developer → API Keys
curl -X GET https://api.mercurypayroll.com.np/v2/employees \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "X-Tenant-ID: your_company_id"
Response 200 OK
{
  "success": true,
  "data": {
    "employees": [...],
    "total": 248,
    "page": 1,
    "per_page": 50
  },
  "meta": {
    "tenant_id": "techco_nepal",
    "fiscal_year": "2081/82"
  }
}
Authentication

🔐 Bearer Token Authentication

All API requests must include your API key as a Bearer token in the Authorization header. Keys can be rotated in Settings → Developer.

# Generate token via login
curl -X POST https://api.mercurypayroll.com.np/v2/auth/token \
  -H "Content-Type: application/json" \
  -d '{
    "email": "admin@techco.com.np",
    "password": "your_password",
    "tenant_id": "techco_nepal"
  }'
Token response
{
  "access_token": "mp_live_eyJhbGci...",
  "token_type": "Bearer",
  "expires_in": 86400,
  "scope": "employees:read payroll:write reports:read"
}

Payroll endpoints
POST/v2/payroll/runTrigger payroll run

⚡ Triggers payroll calculation for the specified month

All salary, TDS, SSF, and PF amounts auto-calculated per Nepal tax rules FY 2081/82.

curl -X POST https://api.mercurypayroll.com.np/v2/payroll/run \
  -H "Authorization: Bearer mp_live_xxx" \
  -d '{
    "month_bs": "2081-12",
    "department": "all",
    "include_overtime": true,
    "dry_run": false
  }'
Response
{
  "payroll_id": "pr_2081_12_techco",
  "employees_processed": 248,
  "total_gross": 38400000,
  "total_tds": 4100000,
  "total_net": 32380000,
  "status": "completed"
}

Webhook events

🔗 Webhooks notify your system when events happen in Mercury Payroll

Register a webhook URL in Settings → Developer → Webhooks. All POST requests include X-Mercury-Signature for verification.

EventTriggered when
payroll.completedMonthly payroll run finishes
payslip.generatedIndividual payslip created
employee.createdNew employee added
tds.filedTDS report exported for IRD
payment.disbursedSalary transferred to bank
leave.approvedLeave request approved
subscription.renewedSaaS plan auto-renewed
Webhook payload example
{
  "event": "payroll.completed",
  "timestamp": "2082-01-02T10:30:00Z",
  "tenant_id": "techco_nepal",
  "data": {
    "payroll_id": "pr_2081_12_techco",
    "employees_paid": 248,
    "total_net": 32380000
  }
}
